HTML Basics

HTML is the backbone of the web, defining structure and meaning.

Elements are represented by tags like <p> for paragraphs.

Attributes provide extra information, e.g. class or id.

HTML documents start with <!DOCTYPE html>.

Semantic tags like <header> and <footer> improve accessibility.

  • Headings define hierarchy
  • Paragraphs hold text
  • Lists organize items
CSS Selectors

CSS styles HTML elements using selectors.

Class selectors use .classname.

ID selectors use #idname.

Universal selector is *.

Attribute selectors target elements with specific attributes.

  • Element selector: p { }
  • Class selector: .box { }
  • ID selector: #main { }
Responsive Design

Responsive design ensures websites look good on all devices.

Media queries adjust styles based on screen size.

Fluid grids use percentages instead of fixed units.

Flexible images scale with their containers.

Mobile-first approach starts with small screens.

  • Use @media rules
  • Design for accessibility
  • Test across devices
JavaScript Fundamentals

JavaScript adds interactivity to web pages.

Variables store data using let, const, or var.

Functions encapsulate reusable logic.

Events respond to user actions.

Objects and arrays organize data.

  • Loops iterate over data
  • Conditionals control flow
  • DOM manipulation changes page content
Version Control

Version control tracks changes in code.

Git is the most popular version control system.

Repositories store project history.

Branches allow parallel development.

Commits record snapshots of code.

  • Initialize with git init
  • Stage changes with git add
  • Commit with git commit
